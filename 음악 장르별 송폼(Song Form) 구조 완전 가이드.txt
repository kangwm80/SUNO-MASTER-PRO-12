              음악 장르별 송폼(Song Form) 구조 완전 가이드
         — 장르마다 다른 곡 구성 공식을 한눈에 —

[ 범례 ]
  I  = Intro (인트로)
  V  = Verse (버스)
  P  = Pre-Chorus (프리코러스)
  C  = Chorus (코러스)
  B  = Bridge (브릿지)
  O  = Outro (아웃트로)
  R  = Rap Verse (랩 버스)
  S  = Solo (악기 솔로)
  D  = Drop (드랍)
  BU = Build-Up (빌드업)
  VR = Vamp/Repeat (반복 섹션)
  IN = Interlude (간주)
  [ ] = 선택사항 (없어도 됨)
  → = 흐름 방향

A. 팝 계열 (Pop Family)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

◆ 메인스트림 팝 (Mainstream Pop)
──────────────────────────────────────────────────────────────────
  표준형:  I → V1 → P → C → V2 → P → C → B → C → [C] → O
  단순형:  I → V1 → C → V2 → C → B → C → O
  후크형:  C → V1 → C → V2 → C → B → C → O  ← 코러스로 시작

  마디수 기준:
    인트로   :  4 ~ 8 마디
    버스     : 16 마디
    프리코러스:  4 ~ 8 마디
    코러스   :  8 ~ 16 마디
    브릿지   :  8 마디
    아웃트로 :  4 ~ 8 마디

  총 길이: 3분 ~ 3분 30초 (스트리밍 최적)

  실전 예시 (Uptown Funk — 115 BPM):
    I → V1 → C → V2 → C → B → C → VR → O

──────────────────────────────────────────────────────────────────
◆ 팝 발라드 (Pop Ballad)
──────────────────────────────────────────────────────────────────
  표준형:  I → V1 → [P] → C → V2 → [P] → C → B → C(전조) → O
  서정형:  I → V1 → C → V2 → C → B → C → C → O

  특징:
    · 코러스 전 전조(Key Change) 1회 권장 → 감동 극대화
    · 마지막 코러스는 악기 줄이고 보컬만으로 시작 후 폭발
    · 브릿지 이후 최소 반박 침묵(rest) → 감정 고조 효과
    · 아웃트로는 코러스 멜로디 반복 + 페이드아웃

  총 길이: 4분 ~ 5분

  실전 예시 (Hello — Adele, 80 BPM):
    I → V1 → C → V2 → C → B → C(반음 UP) → O

──────────────────────────────────────────────────────────────────
◆ K-POP
──────────────────────────────────────────────────────────────────
  표준형:  I → V1 → P → C → V2(랩) → P → C → B(랩/댄스브레이크)
           → C → [포인트코러스] → O

  변형형(최근 트렌드):
    I → V1 → P → C → R1 → C → V2 → P → C → B → C → O

  특징:
    · 랩 버스와 보컬 버스가 교차
    · 브릿지 대신 "댄스 브레이크" 구간 삽입 가능
    · 포인트 코러스: 마지막 코러스에 안무 포인트용 짧은 훅 추가
    · 인트로가 독립된 훅 역할을 하기도 함
    · 아웃트로에 추가 보컬 애드리브 삽입

  총 길이: 3분 ~ 3분 30초

  실전 예시 (BTS Butter — 110 BPM):
    I → V1 → P → C → R → P → C → B → C → O

──────────────────────────────────────────────────────────────────
◆ J-POP / 시티팝 (City Pop)
──────────────────────────────────────────────────────────────────
  표준형:  I → V1 → C → V2 → C → B → C → O
  시티팝형: I(긴 인트로) → V1 → C → V2 → C → IN → C → O

  특징:
    · 인트로가 길고 악기 중심 (분위기 설정 중요)
    · 간주(Interlude)에 악기 솔로 구간 삽입
    · 코러스가 짧고 반복적
    · 아웃트로가 인트로 멜로디 반복 ("순환" 구조)

  총 길이: 4분 ~ 5분

──────────────────────────────────────────────────────────────────
◆ 인디팝 / 드림팝 / 베드룸팝
──────────────────────────────────────────────────────────────────
  자유형:  I → V1 → C → V2 → C → [B] → C → O
  미니멀형: V1 → V2 → C → V3 → C → O  ← 코러스 없이 시작 가능
  분위기형: I(긴) → V1 → [P] → C → V2 → C → O

  특징:
    · 구조가 느슨해도 허용 (분위기가 구조를 대신함)
    · 코러스와 버스의 다이나믹 차이가 크지 않을 수 있음
    · 반복적이고 최면적인 마지막 섹션(fade-out vamp)

  총 길이: 2분 30초 ~ 4분 (자유)

──────────────────────────────────────────────────────────────────
◆ 싱어송라이터 (Singer-Songwriter)
──────────────────────────────────────────────────────────────────
  AABA형:  V1 → V2 → B → V3  (전통 32마디 팝 폼)
  버스형:  V1 → V2 → V3 → ... (코러스 없음, 리프레인으로 통일)
  표준형:  I → V1 → C → V2 → C → B → C → O

  특징:
    · 리프레인(Refrain): 각 버스 끝에 반복되는 1~2줄
      코러스 대신 이 한 줄이 훅 역할을 함
    · 가사와 스토리가 중심. 구조는 그 다음.
    · 어쿠스틱 기반이므로 다이나믹이 섬세함

  실전 예시 (Bob Dylan, The Beatles 스타일):
    V1(+Refrain) → V2(+Refrain) → B → V3(+Refrain)

──────────────────────────────────────────────────────────────────
◆ 유로팝 / 라틴팝 (EuroPop / Latin Pop)
──────────────────────────────────────────────────────────────────
  표준형:  I → V1 → [P] → C → V2 → [P] → C → B → C → O
  댄스팝형: I → V1 → C → V2 → C → IN → C → C → O

  특징:
    · 훅이 매우 단순하고 강렬 (비영어권도 따라 부를 수 있게)
    · 간주에 악기 훅 반복
    · 코러스 마지막에 추가 훅 라인 삽입

  실전 예시 (Despacito — 89 BPM):
    I → V1 → C → V2 → C → B(라틴 랩) → C → O

B. 록 / 메탈 계열 (Rock / Metal Family)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

◆ 클래식 록 / 하드 록 (Classic Rock / Hard Rock)
──────────────────────────────────────────────────────────────────
  표준형:  I → V1 → C → V2 → C → S(기타솔로) → C → O
  확장형:  I → V1 → P → C → V2 → P → C → S → B → C → O

  특징:
    · 기타 솔로(Solo) 구간 필수 (보통 브릿지 위치)
    · 솔로 이후 코러스가 더욱 강렬하게 복귀
    · 인트로에 리프(Riff)가 등장 → 곡의 핵심 정체성
    · 아웃트로가 긴 경우 많음 (페이드아웃 또는 하드 엔딩)

  총 길이: 3분 30초 ~ 5분

  실전 예시 (AC/DC, Led Zeppelin 스타일):
    I(riff) → V1 → C → V2 → C → S(솔로) → C → O(riff)

──────────────────────────────────────────────────────────────────
◆ 얼터너티브 록 / 그런지 (Alternative Rock / Grunge)
──────────────────────────────────────────────────────────────────
  표준형:  I → V1 → C → V2 → C → B → C → O
  대비형:  I(조용) → V1(조용) → C(폭발) → V2(조용) → C(폭발)
           → B(극도로 조용) → C(최대폭발) → O

  특징:
    · "다이나믹 대비"가 핵심: Quiet-Loud-Quiet 공식
    · 버스는 매우 조용하고 억제됨 → 코러스에서 폭발
    · 기타 디스토션을 코러스에서만 ON
    · 브릿지가 코러스보다 더 조용한 경우도 있음

  실전 예시 (Nirvana — Smells Like Teen Spirit):
    I → V1(조용) → P → C(폭발) → V2 → P → C → S → C → O

──────────────────────────────────────────────────────────────────
◆ 펑크 록 / 팝 펑크 (Punk Rock / Pop Punk)
──────────────────────────────────────────────────────────────────
  표준형:  I → V1 → C → V2 → C → B → C → O
  돌진형:  I(짧) → V1 → C → V2 → C → C → O  ← 브릿지 없음

  특징:
    · 빠르고 짧음. 총 길이 2분 ~ 3분이 많음
    · 인트로 거의 없거나 1~2마디로 돌입
    · 에너지가 전체에 걸쳐 일정하게 높음
    · 코러스 반복이 많고 마지막에 크게 외치는 구조

  총 길이: 1분 30초 ~ 3분

──────────────────────────────────────────────────────────────────
◆ 헤비메탈 / 스래시메탈 (Heavy Metal / Thrash Metal)
──────────────────────────────────────────────────────────────────
  표준형:  I(riff) → V1 → C → V2 → C → S(솔로) → C → O
  서사형:  I → V1 → C → V2 → C → B(느린) → S(솔로) → C → O
  공격형:  I(짧) → V1 → V2 → C → V3 → C → S → C → O

  특징:
    · 기타 솔로는 메탈의 필수 요소 (중간 또는 클라이맥스)
    · 리프가 각 섹션의 성격을 정의
    · 코러스가 팝만큼 캐치하지 않을 수 있음
    · 브릿지에서 템포 변화 가능 (느렸다 빠르게)
    · 아웃트로에 리프 반복 + 하드 엔딩(hard stop)

  총 길이: 4분 ~ 6분

──────────────────────────────────────────────────────────────────
◆ 둠 메탈 (Doom Metal)
──────────────────────────────────────────────────────────────────
  자유형:  I(긴) → V1(긴) → [C] → V2(긴) → [B] → O(긴)
  드론형:  I → 반복 리프 → 보컬 섹션 → 반복 리프 → O

  특징:
    · 전통적인 팝 구조를 거부
    · 하나의 분위기와 텍스처가 장시간 지속
    · 코러스가 없거나 매우 단순한 훅
    · 총 길이 8분 ~ 20분 이상도 있음

──────────────────────────────────────────────────────────────────
◆ 심포닉 메탈 (Symphonic Metal)
──────────────────────────────────────────────────────────────────
  표준형:  I(오케스트라) → V1 → P → C(오케스트라+헤비)
           → V2 → P → C → B(오케스트라 인터루드)
           → S(기타솔로) → C(최대) → O

  특징:
    · 오케스트라 인트로와 인터루드가 독립 섹션으로 존재
    · 여성 소프라노 클린 보컬 ↔ 남성 그로울 보컬 교차
    · 코러스에서 오케스트라+기타+보컬 모두 합류
    · 대규모 구성으로 총 길이 5분 ~ 8분

──────────────────────────────────────────────────────────────────
◆ 프로그레시브 록 / 메탈 (Progressive Rock / Metal)
──────────────────────────────────────────────────────────────────
  자유형:  각 섹션이 고정된 구조 없이 전개됨
  개념형:  Part 1 → Part 2 → Part 3 ... (악장 구분)
  확장형:  I → V1 → C → IN(악기) → V2 → B(긴) → S → C → O(긴)

  특징:
    · 박자 변환 필수: 4/4 → 7/8 → 5/4 → 6/8 등
    · 주제(테마)의 발전과 변형이 구조를 대신함
    · 개념 앨범(Concept Album)의 경우 앨범 전체가 하나의 구조
    · 총 길이 7분 ~ 25분

  실전 예시 (Dream Theater, Tool 스타일):
    I(긴) → Part1(테마제시) → Part2(발전) → Part3(클라이맥스)
    → Part4(해소) → O

C. 힙합 / R&B 계열 (Hip-Hop / R&B Family)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

◆ 트랩 / 메인스트림 힙합 (Trap / Hip-Hop)
──────────────────────────────────────────────────────────────────
  표준형:  I → R1 → C(훅) → R2 → C(훅) → R3 → C(훅) → O
  단순형:  I → R1 → C → R2 → C → O
  팝랩형:  I → V1 → C → R1 → C → R2 → C → B → C → O

  힙합 고유 명칭:
    · 버스(Bar/Verse) = 랩 구간 (16~32마디)
    · 훅(Hook) = 팝의 코러스에 해당 (8마디)
    · 아웃트로 = 종종 페이드아웃 또는 추가 바

  특징:
    · 랩 버스 16마디 → 훅 8마디 → 랩 버스 반복이 기본
    · 버스3 이후 아웃트로로 마무리하는 경우 많음
    · 브릿지 없는 경우가 많음 (대신 버스3가 브릿지 역할)
    · 훅은 간단하고 반복적. 멜로딕 랩 or 보컬

  총 길이: 2분 30초 ~ 4분

  실전 예시 (Drake, Travis Scott 스타일):
    I(짧) → R1(16bar) → 훅(8bar) → R2(16bar) → 훅 → R3 → 훅 → O

──────────────────────────────────────────────────────────────────
◆ 붐뱁 / 이스트코스트 (Boom Bap / East Coast)
──────────────────────────────────────────────────────────────────
  표준형:  I(샘플루프) → R1 → 훅 → R2 → 훅 → R3 → [훅] → O
  클래식형: 인트로 → V1(16bar) → 훅(4~8bar) → V2 → 훅 → V3 → O

  특징:
    · 샘플 기반 루프가 전체를 지속
    · 훅이 짧고 반복적 (4~8마디)
    · 버스가 16~32마디로 길고 라임 밀도 높음
    · DJ 컷 & 스크래치 인트로 자주 사용

──────────────────────────────────────────────────────────────────
◆ 컨셔스 랩 / 스토리텔링 랩 (Conscious / Storytelling Rap)
──────────────────────────────────────────────────────────────────
  서사형:  I → R1(이야기 시작) → 훅 → R2(전개) → 훅
           → R3(클라이맥스) → [훅] → O(해소)

  특징:
    · 버스 하나가 단막극처럼 내러티브를 전개
    · 훅이 테마를 요약 (소설의 제목처럼)
    · 버스 길이가 길 수 있음 (24~32마디)

──────────────────────────────────────────────────────────────────
◆ 컨템포러리 R&B (Contemporary R&B)
──────────────────────────────────────────────────────────────────
  표준형:  I → V1 → [P] → C → V2 → [P] → C → B → C → O
  그루브형: I → V1 → C → R(랩브릿지) → V2 → C → C → VR → O

  특징:
    · 팝 구조와 유사하나 그루브와 멜리스마가 핵심
    · 랩 브릿지(Rap Bridge): 힙합 래퍼 피처링 구간
    · 아웃트로에 보컬 애드리브(Ad-lib) 섹션 추가
    · 반복 섹션(Vamp)에서 보컬 즉흥 허용

  총 길이: 3분 30초 ~ 4분 30초

──────────────────────────────────────────────────────────────────
◆ 소울 / 네오 소울 (Soul / Neo Soul)
──────────────────────────────────────────────────────────────────
  표준형:  I → V1 → C → V2 → C → B → C → VR(Ad-lib) → O
  복음형:  I → V1 → C(합창) → V2 → C → B → C(C&R) → VR → O

  C&R = 콜앤리스폰스 (Call & Response)

  특징:
    · VR(Vamp): 곡 마지막에 루프 반복 + 보컬 즉흥 구간
    · 브릿지가 감정적 전환점의 역할을 강하게 함

    인트로         :  8 ~ 16 마디
    보컬 버스      : 16 마디
    빌드업         :  8 ~ 16 마디 (라이저 + 필터 + 스네어 롤)
    드랍           : 16 ~ 32 마디 (모든 요소 폭발)
    브레이크다운   :  8 ~ 16 마디 (악기 제거 + 분위기 전환)
    두 번째 빌드업 :  8 ~ 16 마디
    두 번째 드랍   : 16 ~ 32 마디
    아웃트로       :  8 ~ 16 마디

  특징:
    · 가사는 드랍 전 보컬 구간에만 집중
    · 드랍 자체가 코러스 역할

G. 클래식 / 오케스트라 계열
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

◆ 소나타 형식 (Sonata Form) — 클래식 핵심 구조
──────────────────────────────────────────────────────────────────
  제시부 → 발전부 → 재현부 → [코다]
  (Exposition → Development → Recapitulation → Coda)

  제시부:   제1주제(T) → 경과구 → 제2주제(D) → 종결구
  발전부:   두 주제를 조성 변화·분열·대위로 전개
  재현부:   제1주제(T) → 제2주제(T, 조성 통일) → 종결구
  코다:     마무리 (선택)

  특징:
    · 교향곡 1악장, 피아노 소나타 등에 사용
    · 팝의 "버스→코러스" 대신 "주제 제시→발전→귀환"

──────────────────────────────────────────────────────────────────
◆ 교향곡 (Symphony) — 전체 구조
──────────────────────────────────────────────────────────────────
  4악장 구조 (고전·낭만):
    1악장: 빠름 (소나타형식, Allegro)
    2악장: 느림 (ABA 또는 변주형식, Andante/Adagio)
    3악장: 중간 (미뉴에트/스케르초, Minuet/Scherzo)
    4악장: 빠름 (론도형식 또는 소나타형식, Allegro)

  3악장 구조 (일부):
    1악장: 빠름 / 2악장: 느림 / 3악장: 빠름

──────────────────────────────────────────────────────────────────
◆ 론도 형식 (Rondo Form)
──────────────────────────────────────────────────────────────────
  ABACABA:  A(주제) → B(에피소드) → A → C(에피소드) → A → B → A

  특징:
    · 주제 A가 계속 돌아오는 구조
    · 교향곡 4악장, 협주곡 피날레에 자주 사용
    · 팝의 코러스 반복 개념과 유사

──────────────────────────────────────────────────────────────────
◆ 변주 형식 (Theme and Variations)
──────────────────────────────────────────────────────────────────
  T → V1 → V2 → V3 → V4 → ... → Vn → [코다]
  (Theme → Variation 1 → Variation 2 ...)

  특징:
    · 하나의 주제가 리듬·화성·빠르기 변화로 변형 반복
    · 각 변주는 독립적인 성격을 가짐

──────────────────────────────────────────────────────────────────
◆ 에픽 오케스트라 / 트레일러 뮤직
──────────────────────────────────────────────────────────────────
  표준형:
    I(조용) → BU1(현악 상승) → 클라이맥스1(모든 악기)
    → 브레이크다운(조용+멜로디) → BU2(합창+오케스트라)
    → 클라이맥스2(최대) → O

  구체적 구조:
    0:00 → 조용한 도입 (피아노 or 현악)
    0:30 → 브라스 등장, 긴장감 고조
    1:00 → 1차 클라이맥스 (전 악기)
    1:30 → 브레이크다운 (감성 멜로디)
    2:00 → 2차 빌드업 (합창 추가)
    2:30 → 2차 클라이맥스 (최대 웅장함)
    3:00 → 해소 및 마무리

  총 길이: 2분 ~ 4분 (트레일러 용도)

──────────────────────────────────────────────────────────────────
◆ 실내악 / 현악 사중주 (Chamber Music)
──────────────────────────────────────────────────────────────────
  4악장 소나타:
    1악장(소나타형식) → 2악장(서정형) → 3악장(스케르초)
    → 4악장(론도 또는 소나타)

──────────────────────────────────────────────────────────────────
◆ 피아노 소나타 / 독주곡
──────────────────────────────────────────────────────────────────
  소나타형:  제시부(A-B) → 발전부 → 재현부(A-B) → [코다]
  전주곡형:  자유로운 단악장 구조 (Chopin Preludes 등)
  녹턴형:   ABA' (서정적 세부분 형식)

H. 글로벌 특수 장르
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

◆ 인도 클래식 힌두스탄 (Hindustani Classical)
──────────────────────────────────────────────────────────────────
  알랍-조르-잘라-반디쉬 형식:
    알랍(Alap)      : BPM 없음 / 드론만 / 라가 탐색 (느린 즉흥)
    조르(Jor)       : 리듬 도입 / 아직 탈라 없음
    잘라(Jhala)     : 빠른 리듬 / 현 퍼커션 효과
    반디쉬(Bandish) : 탈라+라가의 작곡된 곡 / 점층적으로 빨라짐

  특징:
    · 고정된 마디 수 없음. 즉흥이 핵심.
    · 전체 연주: 20분 ~ 2시간 (공연에 따라)
    · 라가와 탈라가 감정·시간·계절을 지배

──────────────────────────────────────────────────────────────────
◆ 볼리우드 (Bollywood)
──────────────────────────────────────────────────────────────────
  영화음악형:  I(오케스트라) → V1(남성) → C → V2(여성)
               → C → IN(악기) → B → C → O

  댄스넘버형:  I → V1 → 댄스코러스 → V2 → 댄스코러스
               → 인터루드(춤) → 코러스 → O

  특징:
    · 영화 장면과 연동되는 구조
    · 남성/여성 보컬 교차 (듀엣 형식 많음)
    · 인터루드에 악기 솔로 또는 댄스 브레이크
    · 멜리스마 보컬 런(run)이 자유롭게 삽입
    · 총 길이 4분 ~ 6분

──────────────────────────────────────────────────────────────────
◆ 아랍 팝 (Arabic Pop)
──────────────────────────────────────────────────────────────────
  표준형:  I(우드+현악) → V1 → C → V2 → C → IN(악기)
           → B → C → VR(Ad-lib) → O

  특징:
    · 우드(Oud) 또는 현악 인트로로 마캄 분위기 설정
    · 인터루드에 우드 솔로 또는 카눈(Qanun) 즉흥
    · VR에서 보컬 멜리스마 즉흥 (마왈 스타일)
    · 코러스는 반복적이고 캐치하게

──────────────────────────────────────────────────────────────────
◆ 샹송 (French Chanson)
──────────────────────────────────────────────────────────────────
  서사형:   I → V1 → [R] → V2 → [R] → V3 → [R] → O
  팝형:     I → V1 → C → V2 → C → B → C → O

  R = 르프랭(Refrain, 반복구) — 각 버스 끝에 등장하는 반복 구절

  특징:
    · 서사형이 전통 샹송 (각 버스가 이야기를 전진)
    · 르프랭(Refrain)이 훅 역할
    · 가사가 주인공이므로 멜로디는 단순
    · 극적인 드라마투르기 (이야기의 기승전결)

──────────────────────────────────────────────────────────────────
◆ 삼바 / 브라질 팝 (Samba / Brazilian Pop)
──────────────────────────────────────────────────────────────────
  전통 삼바:  I(퍼커션) → 코플레(V) → 브라상(C) → 코플레 → 브라상 → O
  모던 팝:    I → V1 → C → V2 → C → B → C → O

  특징:
    · 코플레(Couplet): 이야기 버스
    · 브라상(Brasão): 반복 코러스 구절
    · 카니발 삼바엔레두(Samba-enredo): 학교별 퍼레이드 주제가
      단일 주제로 6~8분 지속

──────────────────────────────────────────────────────────────────
◆ 가스펠 (Gospel)
──────────────────────────────────────────────────────────────────
  표준형:  I → V1 → C(합창) → V2 → C → B → C → VR(C&R) → O
  예배형:  I(찬송) → V1 → C × 반복 → VR(Ad-lib+C&R) → O

  특징:
    · VR에서 콜앤리스폰스(C&R)와 Ad-lib가 핵심
    · 코러스가 점층적으로 강해짐 (레이어 추가)
    · 총 길이 4분 ~ 7분 (예배 용도로 길어짐)
    · 감정이 최고조에 이를 때 자연스럽게 마무리

──────────────────────────────────────────────────────────────────
◆ 켈틱 음악 (Celtic Music)
──────────────────────────────────────────────────────────────────
  전통 리엘(Reel)/지그(Jig):
    A부분(8마디) → A반복 → B부분(8마디) → B반복 → (반복)

  민요형:  V1 → C → V2 → C → V3 → C → O
  발라드형: V1 → V2 → V3 → V4 → O  (코러스 없음, 서사형)

  특징:
    · 리엘/지그: AABB 구조로 세트(set) 구성
    · 민요: 각 버스가 이야기 한 절
    · 피날레가 빠르고 강렬하게 마무리
