【3】 노래 제목의 7가지 법칙
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 짧고 강렬하게 (3단어 이하)
   → 단순할수록 검색되고, 기억된다.
   → 예: Hello / Dynamite / Sorry / Butter / Shape of You

2. 감정 키워드 직접 사용
   → 보편적 감정 단어가 즉시 공감을 이끈다.
   → 예: I Will Always Love You / Happy / Hurt

3. 질문형 / 명령형 제목
   → 호기심 자극. 리스너를 곡 안으로 끌어당김.
   → 예: What Do You Mean? / Can't Stop the Feeling / Let It Go

4. 코러스 속 제목 = 훅
   → 제목이 코러스에 반복 → 곡명이 자동으로 기억됨.
   → 예: Uptown Funk / Old Town Road / Bad Guy

5. 역설 / 반전 표현
   → 상식을 비트는 표현이 호기심 자극.
   → 예: Blinding Lights / Good as Hell / Beautiful Trauma

6. 구체적 이미지 / 장면
   → 추상보다 구체적 장면이 시각적 기억을 만든다.
   → 예: Old Town Road / Candle in the Wind / Flowers

7. 발음이 리드미컬하고 시원함
   → 입에서 굴러가는 느낌. 멜로디와 강세가 자연스럽게 맞아야.
   → 예: Despacito / Macarena / Dynamite / Butter

【4】 프로 작사가의 꿀팁 모음
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▶ 훅(Hook)이 전부다
  · 5초만 들어도 뇌에 저장되는 멜로디+가사 조합이 목표.
  · 짧고 단순, 반복해도 질리지 않게.

▶ 가사보다 멜로디가 먼저 (프로 현장 순서)
  · 컨셉 → 트랙 제작 → 멜로디 완성 → 마지막에 작사
  · 멜로디의 리듬·강세에 음절수를 맞추는 것이 핵심.

▶ 보편적 주제 + 구체적 표현
  · "그리워" → "네 향수 냄새가 아직 이 방 안에"
  · 감정보다 장면, 관념보다 이미지.

▶ 클리셰 피하기
  · moon/June, fire/desire 같은 뻔한 라임 금물.
  · 일상적 말투를 변형해 신선하게.

▶ 라임은 도구, 목적이 아님
  · 완전 라임 + 반라임(Near Rhyme) 조합이 세련됨.
  · 억지 라임은 가사를 망친다.

▶ 감정 아크 설계
  · 버스1(시작) → 버스2(심화) → 브릿지(전환) → 코러스(해소)
  · 감정이 고조되어야 청중이 코러스에서 카타르시스를 느낌.

▶ 구어체 우선
  · 작성한 가사를 큰 소리로 읽어보기.
  · 어색하면 바꾸기. 자연스럽게 말하듯 써야 히트.

【5】 비전공자가 반드시 알아야 할 음악 기초 용어
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

· BPM (Beats Per Minute)
  → 1분당 비트 수. 발로 박자 맞출 때 1분에 몇 번 발을 구르는지.
  → 높을수록 빠른 곡. 낮을수록 느린 곡.

· 마디 (Bar / Measure)
  → 음악의 기본 단위. 보통 4박자 = 1마디.
  → 버스는 보통 16마디, 코러스는 8마디.

· 박자 (Time Signature)
  → 4/4박자: 1마디에 4박 (가장 흔한 팝 형식)
  → 3/4박자: 1마디에 3박 (왈츠 느낌, 탱고 일부)
  → 6/8박자: 흔들리는 리듬감 (컨트리, 일부 발라드)

· 음절 (Syllable)
  → 발음의 기본 단위. 글자수가 아니라 발음 단위로 셈.
  → "사랑해" = 3음절 / "beautiful" = 3음절 (beau-ti-ful)
  → 가사 쓸 때는 음절수를 멜로디에 맞춰야 함.

· 훅 (Hook)
  → 귀에 꽂히는 가장 캐치한 멜로디+가사.
  → 대부분 코러스에 위치. 듣고 나면 머릿속에서 자꾸 반복되는 것.

· 라임 (Rhyme / 운율)
  → 줄 끝 발음을 맞추는 것.
  → 완전 라임: night/right/tight / 반라임: time/line

· 플로우 (Flow)
  → 랩·보컬이 비트 위에 얹히는 리듬 패턴.
  → 같은 BPM이어도 음절 배치 방식에 따라 느낌이 완전히 달라짐.

· 멜리스마 (Melisma)
  → 한 음절에 여러 음을 이어 부르는 기법.
  → R&B·발라드·복음·인도 음악에서 많이 사용.
  → 예: "love~♩♪♩♪" — 'love' 한 글자를 여러 음으로 늘임.

· 다이나믹스 (Dynamics)
  → 소리의 크기 변화. 버스(↓작게) → 코러스(↑크게) 흐름이 기본.

· 전조 (Key Change / Modulation)
  → 곡 중간에 조성(키)이 바뀌는 것.
  → 마지막 코러스 전 반음~1음 올리면 감동 극대화 효과.

· 모음의 중요성 (Vowel Placement)
  → 고음 파트에는 입이 크게 벌어지는 모음이 유리.
  → 유리: 아(a), 오(o), 에(e) / 불리: 이(i), 위(ü)
  → 코러스 최고음에 '아/오' 계열 모음 배치 권장.

· 내부 라임 (Internal Rhyme)
  → 줄 중간에도 라임이 등장하는 것.
  → 예: "날 [버리고] 떠난 넌 이미 [잊혔고]"

· 컴핑 (Comping)
  → 반주 악기가 보컬 사이사이를 채우는 것. 재즈에서 중요 개념.

  "BPM + 가사 배치 방식(음표 분할) + 언어 음절 특성"이 결합되어
  실제 쓸 수 있는 음절수의 한계를 결정한다.

─────────────────────────────────────────────────────────────────
음표 분할에 따른 음절 수 계산 (BPM 120, 4/4박자 기준)
─────────────────────────────────────────────────────────────────
  · 4분음표 (기본 비트):  1마디 = 4음절
  · 8분음표 (비트 반으로): 1마디 = 8음절
  · 16분음표 (랩/속사포):  1마디 = 16음절
  · 쉼표·긴 음표 사용:    음절수 ↓ 감소

【7】 글로벌 장르별 완전 작사 가이드
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A. 팝 계열 (Pop Family)
════════════════════════════════════════════════════════════════════

◆ 메인스트림 팝 (Mainstream Pop) — BPM: 100~130
──────────────────────────────────────────────────
목적: 라디오·스트리밍 차트 장악 / 최대 공감 / 따라 부르기

구조: 인트로→버스1→프리코러스→코러스→버스2→코러스→브릿지→코러스

음절 밀도: 16마디 버스에 80~120음절
코러스: 30~50음절 (단순하고 강하게)

핵심 작사 전략:
  1. 코러스 첫 줄 = 제목 = 훅. 3~6단어로 압축.
  2. 쉽고 긍정적인 단어. 영어 단어 믹스 유행.
     예: "너와 함께라면 I'll be OK"
  3. 프리코러스에서 긴장감 고조 (바이럴 포인트).
  4. 브릿지에서 새로운 감정 각도 제시.
  5. 마지막 코러스 전 짧은 침묵 또는 드롭.

무드 키워드: catchy, upbeat, feel-good, romantic, playful,
            emotional, empowering, nostalgic, bittersweet

추천 주제: 설렘·두근거림 / 이별 후 성장 / 자신감·긍정
           사랑의 복잡한 감정 / 청춘·자유

──────────────────────────────────────────────────
◆ 팝 발라드 (Pop Ballad) — BPM: 55~80
──────────────────────────────────────────────────
목적: 감동 / 눈물 유발 / 오래 기억되는 명곡

음절 밀도: 16마디에 40~70음절 (여유롭게)

핵심 작사 전략:
  1. 구체적 장면 묘사가 감동을 만든다.
     나쁜 예: "너를 보고 싶다"
     좋은 예: "아직도 네 번호는 지우지 못했어"
  2. 시적 표현 허용. 은유·비유 적극 사용.
  3. 멜리스마 유도 단어 선택 (모음으로 끝나는 단어).
     예: love / oh / yeah / 사랑해 / 바라봐
  4. 코러스 첫 줄은 10음절 이하, 감정 폭발점.
  5. 브릿지에서 키 변환(전조)으로 감동 극대화.

무드 키워드: romantic, yearning, bittersweet, heartbroken,
            tender, soaring, cinematic, devotional

추천 주제: 이별 후 그리움 / 첫사랑 회상 / 가족 사랑 / 화해

──────────────────────────────────────────────────
◆ K-POP — BPM: 100~150
──────────────────────────────────────────────────
목적: 퍼포먼스 + 훅 + 글로벌 접근성

구조: 인트로→버스1→프리→코러스→버스2→코러스→랩파트→코러스
      (랩 파트가 별도로 있는 경우가 많음)

음절 밀도: 코러스 30~50음절 / 랩 파트 100~180음절

핵심 작사 전략:
  1. 한국어+영어 믹스가 글로벌 접근성을 높임.
     예: "Shining like a diamond / 빛나는 너의 별빛"
  2. 퍼포먼스 포인트 단어: 강한 자음·반복 의성어.
     예: Dynamite / Butter / FIRE / BOOM
  3. 코러스는 영어 단어 중심으로 국제 청중 고려.
  4. 랩 파트는 플로우 변화와 내부 라임 필수.
  5. 훅 직전 "yeah / 아이야 / 오" 등 감탄사 삽입.

무드 키워드: energetic, confident, empowering, playful,
            dreamy, romantic, fierce, youthful

추천 주제: 자신감/당당함 / 팬과의 연결 / 설렘 / 사회적 메시지

──────────────────────────────────────────────────
◆ J-POP & 시티팝 (City Pop) — BPM: 90~120
──────────────────────────────────────────────────
목적: 세련된 감성 / 80년대 레트로 무드 / 일상적 서정

음절 밀도: 중간. 일본어 특성상 모음이 많아 자연스럽게 채워짐.

핵심 작사 전략:
  1. 일상의 구체적 장면과 감각 묘사 (도시, 밤, 빛, 여름).
  2. 직접적 감정 표현보다 상황 묘사로 감정 전달.
  3. 리듬감 있는 일본어 특유의 음절 패턴 활용.
  4. 영어 후렴구 삽입으로 레트로 팝 감성 강화.
  5. 코드 진행과 멜로디가 주인공인 장르 (가사는 보조).

무드 키워드: dreamy, nostalgic, summer-night, groovy,
            city-lights, breezy, warm, retro-elegant

──────────────────────────────────────────────────
◆ 인디팝 / 드림팝 / 베드룸팝 — BPM: 80~120
──────────────────────────────────────────────────
목적: 내면의 진솔한 감정 / 개인적·독립적 표현

핵심 작사 전략:
  1. 의식의 흐름(stream of consciousness) 스타일 허용.
  2. 불완전하고 취약한 감정을 날것으로 표현.
  3. 클리셰를 거부하고 개인적 경험을 구체적으로.
  4. 구조가 느슨해도 됨 (훅이 없어도 분위기가 훅).
  5. 소음·흐릿함(lo-fi)도 미학적 선택으로 활용.

무드 키워드: intimate, hazy, vulnerable, nostalgic, lo-fi,
            indie, DIY, bedroom, introspective, dreamy

추천 주제: 성장통 / 정체성 / 자아 탐색 / 잔잔한 일상


════════════════════════════════════════════════════════════════════
B. 록 / 메탈 계열 (Rock / Metal Family)
════════════════════════════════════════════════════════════════════

◆ 클래식 록 / 하드 록 (Classic Rock / Hard Rock) — BPM: 110~160
──────────────────────────────────────────────────
목적: 에너지 / 반항 / 자유 / 강렬한 감정 표출

구조: 팝 구조와 유사하지만 기타 솔로 구간 추가
     버스→프리→코러스→버스→코러스→기타솔로→코러스

음절 밀도: 중간. 에너지 중심이므로 가사보다 연주가 많은 시간 차지.

핵심 작사 전략:
  1. 강하고 직접적인 언어. 반항·저항·자유 테마.
  2. 코러스는 크게 외칠 수 있는 단어 선택.
     예: "We will rock you!" / "Born to run!"
  3. 이미지가 강렬해야 함. 불, 번개, 폭풍 등 물리적 이미지.
  4. 기타 솔로 구간에는 가사 없음 (악기가 가사 역할).
  5. 브릿지에서 조용해졌다 마지막에 폭발하는 구조 유리.

무드 키워드: raw, powerful, rebellious, gritty, anthemic,
            driving, electric, bold, cathartic, fierce

추천 주제: 반항과 자유 / 길 위의 삶 / 사랑과 욕망 / 시스템 거부

──────────────────────────────────────────────────
◆ 얼터너티브 록 / 그런지 / 인디 록 — BPM: 90~150
──────────────────────────────────────────────────
목적: 진정성 / 복잡한 감정 / 상업성 거부

핵심 작사 전략:
  1. 불완전하고 모순된 감정을 그대로 표현.
  2. 은유와 추상적 이미지를 공격적으로 사용.
  3. 조용한 버스 → 폭발하는 코러스 (다이나믹 대비).
     예: Nirvana의 "Smells Like Teen Spirit"
  4. 사회 비판·소외·정체성 혼란 테마 유효.
  5. 완벽한 라임보다 진정성이 우선.

무드 키워드: angsty, melancholic, restless, raw, distorted,
            cathartic, introspective, alienated, intense

──────────────────────────────────────────────────
◆ 메탈 (Metal) — BPM: 130~220+
──────────────────────────────────────────────────
목적: 극한의 에너지 / 카타르시스 / 기술적 표현

세부 장르별 BPM 및 특성:
  · 그루브 메탈:     100~120 BPM (리프 중심, 헤드뱅잉)
  · 헤비메탈:        130~160 BPM (파워코드, 파워 보컬)
  · 스래시 메탈:     150~200 BPM (속주, 공격적 가사)
  · 둠 메탈:          40~100 BPM (느리고 무거운 분위기)
  · 데스코어:        150~220 BPM (그로울, 스크리밍)
  · 뉴 메탈:         90~140 BPM (힙합+메탈 혼합)

핵심 작사 전략:
  1. 강렬한 이미지·어둠·저항이 핵심.
  2. 그로울(low scream)과 클린 보컬 교차 활용.
     → 클린 보컬 파트: 서정적, 코러스 위치
     → 그로울 파트: 공격적, 버스나 브릿지 위치
  3. 음절이 두꺼운 자음 조합 단어 선택 (발음이 강해야 함).
     예: "Crush / Wreck / Bleed / Shatter / Devour"
  4. 메탈 가사는 서사(narrative)가 있을 때 강력함.
     배경·캐릭터·갈등·해소를 설계.
  5. 철학·역사·신화·전쟁 테마도 자주 사용.

무드 키워드: crushing, relentless, aggressive, cathartic,
            brutal, dark, intense, triumphant, epic, furious

추천 주제: 내면의 악마 / 사회 비판 / 전쟁과 영웅 /
           죽음과 재생 / 반란 / 고통과 초월

──────────────────────────────────────────────────
◆ 심포닉 메탈 (Symphonic Metal) — BPM: 100~160
──────────────────────────────────────────────────
목적: 클래식의 웅장함 + 메탈의 파워 = 서사적 서사

핵심 작사 전략:
  1. 오페라틱 소프라노 + 헤비 기타 = 가사도 이 대비를 살려야.
  2. 서사적이고 문학적인 가사. 판타지·신화·중세 배경.
  3. 코러스는 클린 보컬로 웅장하게, 버스는 긴 서술 가능.
  4. 라틴어·고어(古語) 구절 삽입으로 신비감 강화.
  5. 오케스트라 인트루드(간주)에 맞는 가사 공백 설계.

대표 아티스트: Nightwish, Epica, Within Temptation, Evanescence

무드 키워드: epic, operatic, majestic, cinematic, mystical,
            dark, tragic, triumphant, orchestral, haunting

──────────────────────────────────────────────────
◆ 프로그레시브 록 / 메탈 (Prog Rock / Metal) — BPM: 가변
──────────────────────────────────────────────────
목적: 음악적 복잡성 / 개념 앨범 / 지적 감동

핵심 작사 전략:
  1. 단순한 팝 구조를 거부. 가사도 복잡한 아이디어 가능.
  2. 개념 앨범(Concept Album): 한 앨범에 하나의 스토리.
  3. 긴 시간(7~20분) 곡에서 가사가 여러 章으로 나뉨.
  4. 시·철학·SF·심리학 등 지적 주제.
  5. 박자 변환(5/4, 7/8 등)에 맞는 비대칭 음절 배치.

대표 아티스트: Pink Floyd, Tool, Dream Theater, Yes

C. 힙합 / R&B 계열 (Hip-Hop / R&B Family)
════════════════════════════════════════════════════════════════════

◆ 트랩 / 힙합 (Trap / Hip-Hop) — BPM: 80~160

가사의 위치:
  · 순수 기악곡: 가사 없음. 음악 자체가 언어.
  · 성악곡 (아리아/리트/가곡): 시(詩)를 바탕으로 한 가사.
  · 합창곡: 라틴어·독일어·이탈리아어·영어 혼재.

핵심 특성:
  1. 팝처럼 코러스 반복 구조 없음.
  2. 소나타 형식: 제시부→발전부→재현부 (주제의 전개).
  3. 악장(Movement): 빠른 악장과 느린 악장의 대비.
  4. 가사(성악)가 있을 경우 시의 언어를 그대로 사용.

Suno AI에서 클래식 만들기 전략:
  1. 장르 태그: "Symphonic Orchestral / Chamber Music"
  2. 가사 없이 음악 설명만으로 생성 가능.
  3. 악기 조합 구체적으로 명시.

  4. 미묘한 반음·장식음(Ornament)에 맞는 모음 선택.

◆ Suno AI 가사 작성 팁 (비전공자용)
═══════════════════════════════════════════

1. 가사 구조 태그를 사용하라
   [Verse 1] / [Pre-Chorus] / [Chorus] / [Bridge] 태그를
   가사 앞에 붙이면 Suno가 구조를 인식함.

   예시:
   [Verse 1]
   Standing at the edge of everything I knew
   The city lights below like stars that never grew

   [Chorus]
   I'll carry you / I'll carry you
   Even when the world forgets your name

   [Bridge]
   What we lost we cannot find
   But what remains was always mine

2. 한국어 가사 팁
   · [Verse 1], [Chorus] 등 영어 태그 사용 권장
   · 한국어 가사도 음절에 맞게 작성
   · BPM에 맞는 음절수 고려
   · 멜리스마 유도 단어는 모음으로 끝나게

3. 멜로디 방향 힌트 포함하기
   · "intimate verses, soaring chorus" — 버스는 작게, 코러스는 크게
   · "soft and spoken verses, explosive belted chorus"

4. 반복 구절 설계
   · 코러스는 동일하게 2~3회 반복
   · [Chorus] 태그를 여러 번 쓰면 자동 반복

5. 언어 명시
   · 영어: 기본
   · 한국어: "Korean lyrics" 또는 한국어로 직접 작성
   · 이중 언어: "Korean and English mix"

□ 감정의 흐름이 버스→코러스→브릿지로 고조되는가?
